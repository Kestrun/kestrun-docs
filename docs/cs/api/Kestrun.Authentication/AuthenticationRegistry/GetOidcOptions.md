# AuthenticationRegistry.GetOidcOptions method

Returns a dictionary of all registered OpenID Connect (OIDC) authentication options keyed by their schema. Only entries whose [`AuthenticationType`](../AuthenticationType) is Oidc and whose stored options are of type [`OidcOptions`](../OidcOptions) are included.

```csharp
public IDictionary<string, OidcOptions> GetOidcOptions()
```

## Return Value
