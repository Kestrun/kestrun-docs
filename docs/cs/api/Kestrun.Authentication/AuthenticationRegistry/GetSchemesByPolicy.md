# AuthenticationRegistry.GetSchemesByPolicy method

Returns a list of authentication schema names that define the specified policy name. This method searches through all registered authentication options and returns the schemas that have a [`ClaimPolicyConfig`](../../Kestrun.Claims/ClaimPolicyConfig) containing a policy with the given name.

```csharp
public IList<string> GetSchemesByPolicy(string policyName)
```

| parameter | description |

| policyName | The name of the policy to search for. |
