# Clear-KrSession

## SYNOPSIS
Clears all items from the session.

## SYNTAX

```
Clear-KrSession [<CommonParameters>]
```

## DESCRIPTION
This function accesses the current HTTP context's session and clears all items stored in it.

## EXAMPLES

### EXAMPLE 1
```powershell
Clear-KrSession
Clears all items from the current session.
```
