# New-KrJWTBuilder

## SYNOPSIS
Creates a new JWT token builder instance.

## SYNTAX

```
New-KrJWTBuilder
```

## DESCRIPTION
This function initializes a new instance of the JwtTokenBuilder class, which is used to construct JWT tokens.

## EXAMPLES

### EXAMPLE 1
```powershell
# Creates a new JWT token builder instance
$builder = New-KrJWTBuilder
```
