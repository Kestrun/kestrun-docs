# ViewStart

Set a default layout for all pages so each page only needs to render its body.

## Full source

File: [`pwsh/tutorial/examples/11.1-RazorPages.ps1`][11.1-RazorPages.ps1]

```powershell
{% include examples/pwsh/11.1-RazorPages.ps1 %}
```

File: [`pwsh/tutorial/examples/Assets/Pages/_ViewStart.cshtml`][Assets/Pages/_ViewStart.cshtml]

```razor
{% include examples/pwsh/Assets/Pages/_ViewStart.cshtml %}
```

## Step-by-step

1. Layout: Set `Layout = "Shared/_Layout"` for all pages in the folder.
2. Override: Individual pages can override this by setting `Layout = null` or another layout.

## Try it

If `_ViewStart.cshtml` is applied, the layout header/nav should appear:

```powershell
curl -i http://127.0.0.1:5000/About
```

## Troubleshooting

| Symptom | Cause | Fix |
|---------|-------|-----|
| Layout not applied | `_ViewStart.cshtml` not discovered | Keep the file in the Razor Pages root folder configured by `Add-KrPowerShellRazorPagesRuntime -RootPath` |
| Wrong layout path | Layout file moved | Update the string to match the location under `Shared/` |
