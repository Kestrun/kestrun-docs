Import common namespaces and enable tag helpers for all pages in the folder.

## Step-by-step

1. Namespaces: Add `System` / `System.Linq` so views can reference common types.
2. HTTP types: Import `Microsoft.AspNetCore.Http`.
3. Tag helpers: Enable built-in MVC tag helpers (for example form helpers) with `@addTagHelper`.

## Troubleshooting

| Symptom | Cause | Fix |
|---------|-------|-----|
| Razor compilation error for missing types | Namespace not imported | Add the required `@using` entry here or fully qualify the type |
| Tag helpers not recognized | Tag helpers not added | Ensure `@addTagHelper *, Microsoft.AspNetCore.Mvc.TagHelpers` is present |
