# PowerShell Documentation

Kestrun's PowerShell docs are organized to help you move from first run to production-ready APIs.

## Start here

- **New to Kestrun?** Start with the [Tutorials](/pwsh/tutorial/) for a guided, runnable path.
- **Need a specific feature?** Jump to the [Guides](/guides/) for topic-focused explanations.
- **Looking for generated API reference?** Browse the PowerShell cmdlet reference under [PowerShell Cmdlets](/pwsh/cmdlets/).

## Recommended learning path

1. Work through the [Hello World tutorial](/pwsh/tutorial/1.introduction/1.Hello-World).
2. Continue through the [Tutorials index](/pwsh/tutorial/) in order to learn routing, responses, middleware, and OpenAPI.
3. Use the [Guides](/guides/) when you need deployment, authentication, logging, or production-oriented depth.

## What you will find here

| Section | Purpose |
| --- | --- |
| [Tutorials](/pwsh/tutorial/) | End-to-end, runnable examples in recommended learning order. |
| [PowerShell Cmdlets](/pwsh/cmdlets/) | Reference documentation for the Kestrun module surface area. |
| [Guides](/guides/) | Cross-cutting concepts such as OpenAPI, authentication, sessions, and deployment. |
